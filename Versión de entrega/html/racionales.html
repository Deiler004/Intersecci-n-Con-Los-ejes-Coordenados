Las funciones racionales son funciones que se expresan como el cociente de dos @@polinomios@@. Su forma general es:

f(x) = P(x) / Q(x)

donde P(x) y Q(x) son polinomios y Q(x) ≠ 0.

Una característica importante de estas funciones es que no están definidas cuando el denominador es igual a cero, ya que no es posible dividir entre cero.

Para encontrar las intersecciones con los ejes coordenados se utilizan dos ideas principales:

La intersección con el eje @@y@@ se obtiene cuando x = 0, siempre y cuando Q(x) ≠ 0.
La intersección con el eje @@x@@ ocurre cuando f(x) = 0, lo cual sucede cuando el @@numerador@@ es igual a cero y el denominador es diferente de cero.
