Intersección Con Los Ejes

A trozos

Definición Funciones A Trozos

Una función a trozos es aquella cuya definición cambia según el valor de la variable independiente. Es decir, la función se define mediante distintas expresiones en diferentes intervalos del dominio.

Entre las funciones a trozos más comunes están:

Funciones definidas por intervalos

Ejemplo:

\[
f(x) =
\begin{cases}
2x^2 & \text{si } x \ge 10 \\
-3x  & \text{si } x < -2
\end{cases}
\]

Funciones escalonadas o por partes:

\[
f(x) =
\begin{cases}
x+1 & \text{si } x \ge 0 \\
1-x  & \text{si } x < 0
\end{cases}
\]

Estas funciones permiten modelar situaciones donde el comportamiento cambia dependiendo de ciertas condiciones, por lo que su gráfica puede presentar cambios de regla en distintos puntos del dominio.

Características generales

• Se forman mediante un número finito de operaciones algebraicas.
• Pueden tener restricciones en su dominio (por ejemplo, no se puede dividir entre cero).
• Sus intersecciones con los ejes se obtienen igualando la función a cero ( ejemplo: \(f(x)=0\) ) o evaluando en \(x=0\), siempre que el valor pertenezca al dominio.

  • Esta definida en intervalos
  • Tienen multiples fórmulas
  • Su dominio es la union de multiples dominios disjuntos
  • Puede ser discontinua
  • Las graficas son compuestas por multiples formas

Caracteristica Principal

La característica principal de una función a trozos es que no se describe con una sola expresión matemática, sino con varias, cada una válida únicamente dentro de un intervalo o condición específica del dominio. En otras palabras, la función “cambia de regla” dependiendo del valor que tome la variable.

Esto significa que el dominio de la función se divide en diferentes partes o intervalos, y a cada uno de ellos se le asigna una expresión distinta. Cada una de estas expresiones representa un “trozo” de la función.

Para calcular el valor de una función a trozos, primero se debe observar el valor de la variable independiente y determinar en cuál intervalo o condición se encuentra. Una vez identificado el intervalo correspondiente, se utiliza la expresión asociada a ese intervalo para realizar el cálculo.

\[
f(x) =
\begin{cases}
x^2 & \text{si } x \ge 0 \\
-x  & \text{si } x < 0
\end{cases} \]

En la anterior funcion se puede observar que la condicion para que \[ f(x)=x^2 \] es que x sea mayor o igual a 0, pero si x es menor a 0, f(x) se definira como  \[ f(x)=-x \]

De esta manera, aunque la función tenga varias expresiones, solo una de ellas se utiliza en cada caso, dependiendo del valor de la variable. Esto permite representar situaciones donde el comportamiento de la función cambia según ciertas condiciones.

Interseccion con eje x

Para encontrar la intersección con el eje x de una función a trozos, se debe determinar el valor de x para el cual la función toma el valor 0.

En este caso usaremos la función:

\[f(x) = 
\begin{cases}
    -x-5 & \text{si } x < 5 \\
    2x+1   & \text{si } x≥5
\end{cases}\]


Pasos:

Tomar la expresión de la función correspondiente al intervalo que se está analizando.

\[f(x) = 
\begin{cases}
    -x-5 & \text{si } x < 5 \\
    2x+1   & \text{si } x≥5
\end{cases}\]

 


Igualar la función a 0 para cada trozo.

\[ 2x+1 = 0 ; -x-5 = 0 \]


Resolver la ecuación obtenida para encontrar el valor de x.

\[ x = \frac{-1}{2} ; x = -5 \]


Verificar que el valor obtenido pertenezca al intervalo o condición de ese trozo de la función.

\[ \text{Si } x = \frac{1}{2} \text{ pero } x ≥ 5 \text{ pero eso no es cierto} \]

\[ \text{Si } x = -5 \text{ pero } x < 5 \text{ note que } -5 < 5 \text{ asi que la interseccion esta en este trozo, asi } h(-5)=0 \]


Si el valor cumple la condición del intervalo, entonces corresponde a una intersección con el eje x.

\[ \text{En el caso del primer trozo } \frac{-1}{2} ≥ 5 \text{ asi que no pertenece a este trozo ya que eso no es cierto} \]

\[ \text{En el caso del segundo trozo} -5 < 5 \text{ lo cual es verdadero, asi que la interseccion con el eje es en este trozo} \]


Este procedimiento debe realizarse para cada uno de los trozos de la función.

Interseccion con el eje y

Para encontrar la intersección con el eje y de una función a trozos, se debe determinar el valor que toma la función cuando \(x = 0\).

En este caso usaremos la función:

\[
h(x)=
\begin{cases}
2x+1 & \text{si } x \ge 5 \\
-x-5 & \text{si } x < 5
\end{cases}
\]

Pasos:

Sustituir \(x = 0\) en la función.

\[
h(0)=
\begin{cases}
2(0)+1 & \text{si } 0 \ge 5 \\
-(0)-5 & \text{si } 0 < 5
\end{cases}
\]

Identificar cuál trozo de la función corresponde al valor de \(x\).

\[
0 \ge 5
\]

pero eso no es cierto

\[
0 < 5
\]

nótese que \(0\) es menor que \(5\), así que se utiliza este trozo de la función

Realizar el cálculo con la expresión correspondiente.

\[
h(0) = -(0) - 5
\]

\[
h(0) = -5
\]

En este caso, \(0 < 5\) es verdadero, así que el valor de la función se obtiene usando el segundo trozo. Por lo tanto, la intersección con el eje \(y\) ocurre cuando \(h(0) = -5\).
